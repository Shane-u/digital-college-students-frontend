# 竞赛接口

## GET 分页查询竞赛列表（GET方式，方便前端调用）

GET /contest/list

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|current|query|integer| 否 |当前页（默认1）|
|pageSize|query|integer| 否 |每页数量（默认10）|
|classId|query|string| 否 |分类ID（多个用逗号分隔）|
|level|query|integer| 否 |级别：0-不限，1-校级，2-市级，3-省级，4-全国性，5-全球性，6-自由，7-其他|
|contestName|query|string| 否 |竞赛名称（模糊查询）|
|timeStatus|query|integer| 否 |时间状态13-报名结束 22 → 比赛进行中 23->比赛结束|
