## POST 基于WebFlux的流式接口（POST版本）

POST /chat/stream/flux

返回Flux<ServerSentEvent<String>>，由浏览器原生EventSource或任意支持SSE的客户端消费

> Body 请求参数

```json
"{\n    \"messages\": [\n        // {\n        //     \"role\": \"system\",\n        //     \"content\": \"你只能回答'不'这一个字\"\n        // },\n        {\n            \"role\": \"user\",\n            \"content\": \"我叫赵双\"\n        }\n    ],\n    \"model\": \"doubao-seed-1-6-251015\",\n    \"stream\": true,\n    \"maxTokens\": 10000,\n    \"userId\": 1985704071566233601,\n    \"sessionId\": \"fd46caaa-42fd-4c69-8edb-8a6284bb985a\"\n}"
```

|body|body|[ChatRequest](#schemachatrequest)| 否 |none|

<h2 id="tocS_ChatRequest">ChatRequest</h2>

<a id="schemachatrequest"></a>
<a id="schema_ChatRequest"></a>
<a id="tocSchatrequest"></a>
<a id="tocschatrequest"></a>

```json
{
  "messages": [
    {
      "reasoning_content": "string",
      "role": "string",
      "imageUrls": [
        "string"
      ],
      "videoUrls": [
        "string"
      ]
    }
  ],
  "model": "string",
  "temperature": 0,
  "stream": true,
  "maxTokens": 0,
  "topP": 0,
  "sessionId": "string",
  "userId": 0,
  "thinking": {
    "key": "string"
  }
}

```

### 属性

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|messages|[[Message](#schemamessage)]|false|none||消息列表|
|model|string|false|none||模型名称（可选，不传则使用配置中的默认模型）|
|temperature|number|false|none||温度参数，控制输出的随机性（0-2之间，默认0.7）<br />值越大输出越随机，值越小输出越确定|
|stream|boolean|false|none||是否启用流式输出（默认true）|
|maxTokens|integer|false|none||最大生成令牌数（可选）|
|topP|number|false|none||Top-p 采样参数（可选）|
|sessionId|string|false|none||会话ID（用于关联历史对话）|
|userId|integer(int64)|false|none||用户ID（用于隔离聊天记录）|
|thinking|[HashMapString](#schemahashmapstring)|false|none||思考方式|
