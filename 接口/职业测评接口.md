# 题目接口

## GET 获取MBTI问卷题目列表

GET /question/mbti

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": [
    {
      "id": 0,
      "title": "",
      "options": [
        {
          "id": 0,
          "title": ""
        }
      ]
    }
  ],
  "message": ""
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[BaseResponseListQuestionVO](#schemabaseresponselistquestionvo)|
