# 成长记录接口

## POST 上传文件（支持 MD5 秒传）

POST /growth-record/upload/file

> Body 请求参数

```yaml
file: ""

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|
|» file|body|string(binary)| 是 |文件|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": {
    "id": 0,
    "fileName": "",
    "fileUrl": "",
    "fileSize": 0,
    "createTime": ""
  },
  "message": ""
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[BaseResponseGrowthFileVO](#schemabaseresponsegrowthfilevo)|

## POST 上传图片（支持 MD5 秒传）

POST /growth-record/upload/image

> Body 请求参数

```yaml
file: file:///Users/shane/Downloads/interview/personal_info/证件照.png

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|type|query|integer| 是 |类型：1-单纯作为照片存储，2-成长记录中的照片|
|uploadTime|query|string| 否 |目标上传时间（用户选择的上传时间，格式：yyyy-MM-dd HH:mm:ss 或 yyyy-MM-dd）|
|body|body|object| 否 |none|
|» file|body|string(binary)| 是 |图片文件|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": {
    "id": 0,
    "imageName": "",
    "imageUrl": "",
    "imageSize": 0,
    "type": 0,
    "uploadTime": "",
    "createTime": ""
  },
  "message": ""
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[BaseResponseGrowthImageVO](#schemabaseresponsegrowthimagevo)|
