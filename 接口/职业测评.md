## POST 启动职业规划工作流

POST /agent/career/start

> Body 请求参数

```json
{
  "userId": 0,
  "input": "string",
  "assessmentResult": {
    "key": {}
  }
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|[StartRequest](#schemastartrequest)| 否 |none|

> 返回示例

> 200 Response

```json
{
  "graph": "/agent/career/graph",
  "runId": "",
  "sse": "/agent/career/stream?runId=",
  "report": "/agent/career/report?runId="
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[MapObject](#schemamapobject)|
