# 爬虫接口

## GET 分页查询招聘信息

GET /crawler/job-info/list/page

支持通过 workName 字段进行模糊匹配

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|workName|query|string| 否 |工作名称（模糊匹配，可选）|
|companyName|query|string| 否 |公司名称（模糊匹配，可选）|
|workAddress|query|string| 否 |工作地址（模糊匹配，可选）|
|current|query|integer| 是 |当前页码，默认1|
|pageSize|query|integer| 是 |每页数量，默认10，最大100|

> 200 Response
