# 数字化大学生前端系统

一个基于Vue3的现代化大学生管理系统，采用科技感蓝色系设计，具有简洁大气的界面和流畅的滑动切换效果。

## 功能特性

- 🎨 **科技感蓝色系设计** - 采用蓝色主色调，简洁大气，符合数字大学生审美
- ✨ **动态粒子背景** - 仿照MagicUI的粒子效果，具有鼠标交互和光晕效果
- 🔄 **平滑滑动切换** - 页面间流畅的滑动过渡效果，增强用户体验
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🎯 **完整功能流程** - 包含登录、注册、忘记密码、学校选择等完整流程
- ⚡ **Vue3 + Vite** - 使用最新的前端技术栈
- 🔐 **多重验证机制** - 图片验证码、密码强度检测等安全功能

## 页面功能

### 登录页面
- **三种登录方式**：
  - 手机号 + 密码
  - 手机号 + 短信验证码
  - 邮箱 + 密码
- 记住我功能
- 跳转到注册和忘记密码页面

### 注册页面
- 邮箱、手机号输入
- 密码强度实时检测
- 图片验证码验证
- 表单完整性验证

### 忘记密码页面
- **两步流程**：
  - 第一步：输入邮箱/手机号 + 图片验证码
  - 第二步：设置新密码
- 步骤指示器显示进度

### 学校选择页面
- 学校搜索功能
- 专业搜索功能
- 年级选择
- 完成后进入系统主页

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vue Router 4** - 官方路由管理器
- **Vite** - 下一代前端构建工具
- **CSS3** - 现代CSS特性，包含动画和渐变
- **Canvas API** - 图片验证码生成和粒子动画
- **WebGL/Canvas** - 高性能粒子背景渲染

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```

## 项目结构

```
src/
├── components/      # 组件
│   ├── CaptchaComponent.vue      # 图片验证码组件
│   └── ParticleBackground.vue    # 粒子背景组件
├── views/           # 页面组件
│   ├── LoginPage.vue    # 登录页面（包含所有子页面）
│   └── HomePage.vue     # 主页
├── router/          # 路由配置
│   └── index.js
├── App.vue          # 根组件
├── main.js          # 入口文件
└── style.css        # 全局样式
```

## 核心功能

### 动态粒子背景
- 仿照MagicUI的粒子效果设计
- 鼠标交互响应（粒子会跟随鼠标移动）
- 粒子间动态连接线
- 光晕和脉冲效果
- 渐变色彩变化
- 高性能Canvas渲染

### 图片验证码
- 使用Canvas API生成
- 包含干扰线和干扰点
- 支持点击刷新
- 与整体风格统一

### 密码强度检测
- 实时检测密码强度
- 可视化强度指示器
- 多维度评估（长度、复杂度等）

### 搜索功能
- 学校名称搜索
- 专业名称搜索
- 实时过滤结果
- 点击选择

## 开发说明

项目使用Vue3的Composition API编写，代码结构清晰，易于维护和扩展。所有样式都采用CSS变量管理，方便主题定制。组件化设计便于复用和维护。
