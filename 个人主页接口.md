# 用户接口

## POST 更新个人信息

POST /user/update/my

> Body 请求参数

```json
{
  "userName": "string",
  "userAvatar": "string",
  "userProfile": "string",
  "gender": "string",
  "grade": "string",
  "major": "string",
  "school": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|[UserUpdateMyRequest](#schemauserupdatemyrequest)| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": false,
  "message": ""
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[BaseResponseBoolean](#schemabaseresponseboolean)|

## POST 头像上传（MinIO）

POST /user/upload/avatar

仿照成长记录上传逻辑，将图片上传到 MinIO 并返回可访问 URL

> Body 请求参数

```yaml
file: file:///Users/shane/Downloads/interview/personal_info/证件照.png

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|
|» file|body|string(binary)| 是 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "data": "",
  "message": ""
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[BaseResponseString](#schemabaseresponsestring)|
